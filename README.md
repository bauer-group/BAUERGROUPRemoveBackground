# BAUERGROUP - RemoveBackground
Das Shopware 5 Plugin stellt die hochgeladenen Bilder automatisch frei. Dazu wird der Dienst https://www.remove.bg/ verwendet. Sie benötigen hierfür einen entsprechenden Account.
